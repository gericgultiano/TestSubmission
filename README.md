## Front End (using vscode)

### Setup
```
  cd Kiosk_FrontEnd
  npm install primevue
  npm install primeicons
```

### To Run
```
  cd Kiosk_FrontEnd
  npm run dev
```

### Setup Cypress
```
cd Kiosk_FrontEnd
npm install cypress
```
### To run
```
cd Kiosk_FrontEnd
npx cypress open
```
